# Data Cleaning and Preprocessing Project

This project was completed during my internship at CEI, where I worked on cleaning and preprocessing a real-world dataset using Python and Pandas. The main objective was to prepare a structured and analysis-ready dataset by applying essential data cleaning techniques and database normalization concepts.

---

## Project Overview

The project included:

- Exploring and understanding the dataset
- Handling missing values
- Removing duplicate records
- Performing basic data filtering and transformation
- Creating derived columns for better analysis
- Applying normalization techniques (1NF, 2NF, 3NF)
- Exporting the final cleaned dataset
